# 🌿 GreenCart — ফুল-স্ট্যাক গ্রোসারি ডেলিভারি অ্যাপ

> Next.js 15 · TypeScript · Tailwind · MongoDB · Firebase Auth · React Query · Zustand

## ⚡ দ্রুত শুরু

### ১. ইনস্টল
```bash
npm install
```

### ২. .env.local পূরণ করুন
(সমস্ত variable .env.local ফাইলে আছে)

### ৩. সিড করুন
```bash
npm run seed
# 8 বিভাগ + 20 পণ্য তৈরি হবে
```

### ৪. অ্যাডমিন সেট করুন
MongoDB-তে আপনার ইউজারের role = "admin" করুন।

### ৫. চালান
```bash
npm run dev
```

## 📄 সম্পূর্ণ ডকুমেন্টেশন

API রেফারেন্স, ফোল্ডার স্ট্রাকচার, এবং ডিপ্লয়মেন্ট গাইড নিচে।
